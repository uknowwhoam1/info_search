基于python3写的信息收集工具，集合了空间搜索,whois等功能

使用了GPL开源协议

Hunter奇安信的api_key存放在key.txt中！

自带字典fuzz.txt可以替换内容，但请不要修改文件名！

目录扫描可能存在误报，目前能识别重定向了
